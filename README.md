# Blog Personal FluxDev 📚

## Descripción

FluxDev Blog es una plataforma personal diseñada para compartir artículos, opiniones y conocimientos sobre desarrollo y tecnología. Ofrece una experiencia de lectura muy fluida donde los usuarios pueden buscar temas específicos y participar dejando comentarios.

## Características

- **Diseño Atractivo**: Interfaz limpia, minimalista y diseñada especialmente para facilitar la lectura sin distracciones.
- **Interacción Comunitaria**: Puedes leer opiniones de otros usuarios y dejar tus propios comentarios al final de cada artículo.
- **Búsqueda Instantánea**: Encuentra rápidamente publicaciones mediante palabras clave.
- **Soporte Multilingüe**: Navega y descubre contenido disponible tanto en español como en inglés.

## Secciones

1. **Inicio**: Presenta los artículos más recientes o destacados para que empieces a leer de inmediato.
2. **Blog / Artículos**: Listado completo de todas las publicaciones, organizado para una fácil navegación.
3. **Etiquetas**: Permite explorar artículos agrupados por temáticas o categorías específicas.
4. **Acerca de**: Pequeña sección con información sobre el autor del blog.

## Uso

- **Visualización del Proyecto:** El proyecto ya se encuentra en funcionamiento. Puedes explorarlo en nuestra plataforma de despliegue en la nube: [FluxDev Blog](https://fluxdev-nebula.mgdc.site/).
- **Explorar Contenido**: Da un vistazo a las portadas de los artículos y sumérgete en el formato de lectura.
- **Interactuar**: Desplázate al final de las publicaciones si deseas usar el formulario de comentarios.


## Tecnologías Utilizadas

- HTML / CSS / JavaScript
- Astro
- Tailwind CSS
- Hono
- SQLite (Base de datos)
- pnpm

## Instalación

1. **Clonar el Repositorio**: Clona este repositorio en tu máquina local utilizando Git.

```bash
git clone https://github.com/Ivandv19/blog-personal-fluxdev.git
```

2. **Instalar Dependencias**: Navega al directorio del proyecto y ejecuta:

```bash
pnpm install
```

3. **Iniciar el Proyecto**: Ejecuta el proyecto localmente con:

```bash
pnpm run dev
```
